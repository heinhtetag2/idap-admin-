# iDap Client — Foundation

Tech foundation for the **iDap Business** portal. Feature-Sliced Design layout, TypeScript, server state, validation, mock API, and typed PRD business rules.

## Quick start

```bash
npm install
npm run dev          # http://localhost:5173
npm run typecheck    # tsc --noEmit
npm run build        # production bundle
```

## Env

Copy `.env.example` → `.env.local`. Defaults:

| Var | Default | Purpose |
|---|---|---|
| `VITE_API_URL` | `https://api.idap.mn` | Backend base URL |
| `VITE_FILES_URL` | `https://files.idap.mn` | Object storage (R2) |
| `VITE_USE_MOCK_API` | `true` | Enable MSW (dev only) |
| `VITE_DEFAULT_LOCALE` | `mn` | Mongolian Cyrillic |

## Architecture

```
main.tsx
  ├─ bootstrap MSW        (dev + VITE_USE_MOCK_API)
  └─ render
       └─ QueryClientProvider
            └─ App  →  RouterProvider  →  pages
```

- **TanStack Query** — global, call `useQuery` / `useMutation` anywhere
- **MSW** — only starts in dev with mock flag on; ignores unhandled requests
- **Zustand / Zod / @dnd-kit** — no provider needed, import where used
- **React Query Devtools** — mounted in dev, collapsed by default

## Import conventions

All cross-layer imports use the `@/*` alias:

```ts
import Dashboard from '@/pages/dashboard';
import { Sidebar } from '@/widgets/sidebar';
import { cn } from '@/shared/lib/cn';
import { Calendar } from '@/shared/ui/calendar';
import { routes } from '@/shared/config/routes';
import { qualityBand } from '@/shared/config/business';
```

FSD dependency rule: a layer may only import from layers below it — `app → pages → widgets → features → entities → shared`.

## Business rules (typed)

`src/shared/config/business.ts` encodes the PRD so reward/trust UI can import directly:

```ts
PLATFORM_FEE          // { defaultPct: 4, minPct: 0, maxPct: 20, fallbackPct: 4 }
REWARD                // { minMnt: 0, maxMnt: 100_000, holdWindowHours: 24 }
WITHDRAWAL            // { minMnt: 10_000, gateways: ['qpay', 'bonum'] }
TRUST_LEVELS          // 1 Newcomer → 5 Partner (with thresholds)

qualityBand(score)    // → 'paid_instant' | 'held_24h' | 'invalidated' | 'flagged'
rewardMultiplier(avg) // → 0.8 | 0.9 | 1.0 | 1.1 | 1.2
```

## Route constants

`src/shared/config/routes.ts` — typed paths for all four portals (`client`, `respondent`, `admin`, `auth`, `marketing`) in one source of truth.
